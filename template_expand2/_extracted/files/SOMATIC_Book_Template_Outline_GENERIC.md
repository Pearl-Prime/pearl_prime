## DESIGN PHILOSOPHY

The existing bestseller template teaches: **"Your brain is running old software. Update the code."**
This somatic template teaches: **"Your body has been trying to talk to you. Learn to listen."**

Both are topic-agnostic. Neither says "anxiety" or "anger" or "confidence." Both address ANY struggle the listener is facing. The difference is the entry point:

- **Existing template:** Mind-first. Reframe thoughts → change behavior → feel different.
- **This template:** Body-first. Notice sensation → regulate the nervous system → think clearly.

The somatic template CHALLENGES the mind-first approach without attacking it. It says: "That advice wasn't wrong. It was incomplete. It started in the wrong place."

### What makes this GENERIC (not topic-specific):
- Chapters teach SOMATIC SKILLS, not self-help topics
- "Whatever you're dealing with" language — the listener fills in their own struggle
- Exercises apply universally — the same breath practice helps whether you're stuck, overwhelmed, numb, or scattered
- No chapter is "the anxiety chapter" or "the relationship chapter"
- The body-based lens works for ALL problems because ALL problems have a nervous system component

### Key phrases that keep it universal:
- "That thing you've been carrying"
- "Whatever brought you here"
- "The pattern you keep running into"
- "That feeling you can't quite name"
- "The version of this you know best"

### Somatic Exercise Injection Strategy
Because chapters are NOT topic-specific, exercise selection at injection time should be based on:

1. **Chapter position in the arc** (intensity progression):
   - Chapters 1–3 (HARDSHIP): Level 1 exercises only
   - Chapters 4–6 (HELP): Level 1 + light Level 2
   - Chapters 7–9 (HEALING): Level 1–2, some Level 2 activating
   - Chapters 10–12 (HOPE): Level 1–2, integration-focused

2. **Section position within chapter**:
   - Section 04 (first exercise): Gentler, entry-level for the chapter's intensity tier
   - Section 08 (second exercise): Deeper, building on section 04

3. **Functional intent** (what the exercise DOES, not what topic it addresses):
   - Grounding exercises → Chapters 1–3 (establishing body contact)
   - Regulating exercises → Chapters 4–6 (learning the gearshift)
   - Releasing exercises → Chapters 7–8 (moving stored energy)
   - Integrating exercises → Chapters 9–12 (daily practice, embodied living)
