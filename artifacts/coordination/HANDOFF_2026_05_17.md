## Architecture decisions made

### 1. R2 over LFS for media (PR #1142, #1143, #1148, #1149)
GitHub LFS budget is **exhausted** on this repo. Every `lfs: true` checkout fails with `batch response: This repository exceeded its LFS budget`. PR #1120 / #1123 cycle confirmed.

Strategy:
- **Small assets (<1 MB)** → un-LFS via path-specific `.gitattributes` override → ship as regular git blobs → served by Cloudflare Pages directly
- **Large assets (>1 MB videos, audio, multi-format covers)** → upload to R2 bucket `phoenix-omega-artifacts` under `teacher_showcase/` prefix → reference via `pub-7c709cd3f8e6425dbeffaed4ca2cf901.r2.dev` public domain
- R2 Public Development URL is the canonical media-serving path now

### 2. CJK cover rendering (PR #1148, #1149)
The bundled `render_kdp_cover.py` only ships Latin fonts (EB Garamond, Inter, Playfair, Caveat) → CJK strings rendered as tofu (□□). Solution:
- Custom PIL compositor (`/tmp/cjk_cover_renderer.py`, `/tmp/compose_square_covers.py`) using:
  - macOS `Hiragino Sans GB.ttc` (covers zh-CN + zh-TW AND most ja kanji including 此)
  - Bundled `assets/fonts/kdp_overlay/noto_serif_jp/NotoSerifJP-Bold.ttf` for Joshin's title (more contemplative feel)
- Stage 1 FLUX schnell renders (via Pearl Star ComfyUI) provide imagery; Stage 2 PIL composite overlays localized title/subtitle/author

### 3. CJK Ch1 translation strategy (PR #1156)
~1400-word per-teacher Ch1 in 4 ja + 2 zh teachers = ~8400 words if translated fully. Discovered the 13 Ch1s share ~90% template (Pearl_PM's `gen_z_professionals × anxiety` wave) with only ~3 per-teacher voice-line inserts.
- 3 high-quality template translations (ja, zh-CN, zh-TW) covering opening + mechanism + Priya scene + closing (~600 words each)
- 6 per-teacher voice-line inserts at 3 placement points

This is a **PREVIEW** (~600 words per non-English teacher) compared to full ~1400-word for English. A polished full-Ch1 translation pass for the 6 non-English teachers remains a future task.

### 4. FLUX schnell over dev on Pearl Star
`flux1-dev-fp8.safetensors` on Pearl Star is **corrupted** (983 KB on disk; should be GBs). `flux1-schnell-fp8.safetensors` works. All Stage-1 imagery this session used schnell. Operator action: re-download dev on Pearl Star when convenient (`huggingface-cli download black-forest-labs/FLUX.1-dev`).

### 5. Author byline pattern
Real authors per brand are listed in `config/catalog_planning/teacher_brand_author_roster.yaml`. Only `stillness_press` (ahjan, 12 authors) and `clear_seeing_books` (joshin, 6 authors) have populated `authors:` arrays. The other 5 teacher-mode brands declare `author_count` but the populated section is empty:
