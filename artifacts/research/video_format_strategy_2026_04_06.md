### 1.4 Key Finding: YouTube Rejects "Reused Content"

YouTube's Partner Program now rejects monetization for videos built solely from reused free assets (stock footage only, no original elements). Adding ANY original element avoids this:
- Custom text overlays
- Original waveform visualization
- AI-generated images (novel, not stock)
- Custom kinetic typography

**Implication:** Our pipeline MUST add at least one original visual layer to every video.

## PART 2: GUIDED MEDITATION & SOMATIC VIDEO FORMATS (FACELESS)

### 2.1 What Works for Meditation Visuals

| Visual Type | Effectiveness | Notes |
|-------------|---------------|-------|
| **Slow nature footage** (forests, water, clouds) | Highest engagement | Universally calming, 40-60% retention |
| **Abstract particles/gradients** | High | Non-distracting, Headspace-inspired |
| **Breathing circle animation** | Very High for exercises | 4s expand / 2s hold / 6s contract (Headspace model) |
| **Soft color shifts** (dawn gradient) | Medium-High | Subtle, non-distracting |
| **Mandalas / sacred geometry** | Medium | Niche appeal, strong in spiritual market |
| **Galaxy/starfield** | High for sleep | Michael Sealey's signature style |
| **Text cues** ("Breathe in...") | Medium-High | Essential for somatic/body scan |
| **Timer/progress bar** | Medium | Helps with session pacing |
| **Candle flame / fireplace** | High for ambient | ASMR crossover audience |

### 2.2 What Does NOT Work

- **Fast-moving visuals** — contradicts relaxation purpose
- **Bright/saturated colors** — overstimulating
- **Complex scene changes** — breaks meditative state
- **Face/person on screen** — creates comparison/distraction for meditation
- **Text-heavy screens** — users close eyes during meditation
- **Busy patterns** — anxiety-inducing for target audience

### 2.4 Headspace vs Calm Visual Language (Industry Standard)

**Headspace Style:**
- Rounded cartoon characters, warm gradients
- No angular geometry; minimal faces (two dots for eyes)
- Breathing circle: expands 4s, contracts 6s (paces parasympathetic activation)
- Palette avoids pure black/white — warm neutrals only
- Every animation serves emotional objective: reduce anxiety, invite stillness

**Calm Style:**
- Realistic flowing nature — rivers, rain, leaves
- Photography-based aesthetic (not illustrated)
- Backgrounds are the meditation itself
- More "premium/mature" feel vs Headspace's "approachable/playful"

**Implication for our pipeline:** We should offer BOTH styles:
- "Illustrated calm" (Headspace-like) for younger/anxious audiences
- "Nature immersive" (Calm-like) for sleep/relaxation content

### 2.5 Somatic Exercise Specific Formats

For body scan, grounding, and somatic practices:
- **Text cues are essential** — users need instruction overlay ("Notice your feet," "Soften your jaw")
- **Breathing circle** — synchronized with narrator's pacing cues
- **Body outline diagram** — highlighting the body part being addressed (optional, high effort)
- **Minimal background** — nature or solid gradient, NOT busy
- **5-minute format** — "somatic shorts" are the emerging trend
- **Vertical 9:16** — mobile-first for exercise content (users follow along on phone)

### 5.2 New Formats to Add

#### Format A: `nature_meditation`
**Description:** Looping nature footage background + optional text cues + optional breathing circle
**Use for:** Guided meditation, sleep content, body scan, NSDR
**Technical approach:**
```
# Base: royalty-free nature loop from Pexels/Pixabay OR AI-generated (Runway/Pika)
# Layer 1: Nature video (loop with -stream_loop -1)
# Layer 2: Semi-transparent text cues (FFmpeg drawtext with timing from SRT)
# Layer 3: Optional breathing circle overlay (pre-rendered GIF, expanding/contracting)
# Layer 4: Audio (TTS narration + ambient music)

ffmpeg -stream_loop -1 -i nature_loop.mp4 \
  -i breathing_circle.gif \
  -i narration.mp3 \
  -filter_complex "[0:v]scale=1920:1080[bg]; \
    [1:v]scale=200:200[circle]; \
    [bg][circle]overlay=x=(W-w)/2:y=H-h-50:shortest=1[v]; \
    [v]drawtext=fontfile=font.ttf:textfile=cues.txt:fontcolor=white@0.8: \
    fontsize=48:x=(w-text_w)/2:y=(h-text_h)/2:enable='between(t,10,15)'[out]" \
  -map "[out]" -map 2:a -shortest output.mp4
```

#### Format B: `waveform_text`
**Description:** Audio waveform visualization + text-on-screen (word-by-word or phrase-by-phrase)
**Use for:** Affirmations, self-help audiobook clips, motivational content
**Technical approach:**
```
# FFmpeg showwaves filter + drawtext with timed SRT
ffmpeg -i audio.mp3 \
  -filter_complex "[0:a]showwaves=s=1920x200:mode=cline:rate=30: \
    colors=0x88CCFF@0.7:scale=sqrt,format=yuva420p[wave]; \
    color=c=0x1a1a2e:s=1920x1080:d=duration[bg]; \
    [bg][wave]overlay=0:H-200[v]; \
    [v]drawtext=fontfile=font.ttf:textfile=script.srt: \
    fontcolor=white:fontsize=64:x=(w-text_w)/2:y=(h-text_h)/3[out]" \
  -map "[out]" -map 0:a output.mp4
```

#### Format C: `kinetic_text`
**Description:** Animated text appearing word-by-word or phrase-by-phrase with emphasis effects
**Use for:** Self-help quotes, affirmations, book key takeaways, Shorts/TikTok
**Technical approach:**
- Use ASS (Advanced SubStation Alpha) subtitle format for rich animation
- FFmpeg drawtext with time-based `enable` expressions per word
- Pre-render text animations as transparent video overlays
- Overlay on solid gradient or subtle particle background

```
# Using ASS subtitles for rich kinetic effects
ffmpeg -i audio.mp3 \
  -f lavfi -i "color=c=0x0d1117:s=1080x1920:d=60,format=yuv420p" \
  -vf "ass=kinetic_script.ass" \
  -map 1:v -map 0:a -shortest output.mp4
```

#### Format D: `breathing_exercise`
**Description:** Breathing circle animation (expand/hold/contract) synchronized with narration + text cues
**Use for:** Somatic exercises, breathing practices, grounding techniques, body awareness
**Technical approach:**
- Pre-generate breathing circle as animated GIF/WebM (ImageMagick or Python + Pillow)
- Circle timing: 4s inhale (expand), 2s hold, 6s exhale (contract) — Headspace model
- Overlay on calm gradient background
- Text cues from SRT file ("Breathe in...", "Hold...", "Let go...")

```python
# Generate breathing circle with Python (Pillow)
from PIL import Image, ImageDraw
import math

def generate_breathing_frames(width=400, height=400, fps=30,
                                inhale_s=4, hold_s=2, exhale_s=6):
    cycle = inhale_s + hold_s + exhale_s  # 12 seconds
    total_frames = int(cycle * fps)
    frames = []
    for i in range(total_frames):
        t = i / fps
        if t < inhale_s:
            # Expanding
            radius = 50 + (150 * (t / inhale_s))
        elif t < inhale_s + hold_s:
            # Hold at max
            radius = 200
        else:
            # Contracting
            progress = (t - inhale_s - hold_s) / exhale_s
            radius = 200 - (150 * progress)

        img = Image.new('RGBA', (width, height), (0, 0, 0, 0))
        draw = ImageDraw.Draw(img)
        cx, cy = width // 2, height // 2
        draw.ellipse([cx-radius, cy-radius, cx+radius, cy+radius],
                     fill=(136, 204, 255, 100), outline=(136, 204, 255, 200))
        frames.append(img)
    return frames
```

#### Format E: `hybrid_nature_text_wave`
**Description:** Nature background + text overlay + small waveform indicator
**Use for:** The "premium" format — guided meditation, audiobook narration, self-help
**Technical approach:**
- Layer 1: Looping nature video (background)
- Layer 2: Semi-transparent dark gradient bar at bottom
- Layer 3: Text overlay (current phrase/sentence)
- Layer 4: Small waveform in corner (visual "audio is playing" indicator)
- Layer 5: Audio (narration + ambient music)

This is the **highest-retention format** based on research and should be our default for YouTube long-form.

## APPENDIX B: FFmpeg Quick Reference

### Waveform visualization
```bash
ffmpeg -i audio.mp3 -filter_complex \
  "[0:a]showwaves=s=1920x200:mode=cline:rate=30:colors=0x88CCFF@0.7:scale=sqrt,format=yuva420p[wave]; \
  color=c=0x1a1a2e:s=1920x1080:d=$(ffprobe -v error -show_entries format=duration -of csv=p=0 audio.mp3)[bg]; \
  [bg][wave]overlay=0:H-200[out]" \
  -map "[out]" -map 0:a -c:v libx264 -crf 23 -c:a aac output.mp4
```

### Nature loop + text overlay
```bash
ffmpeg -stream_loop -1 -i nature_loop.mp4 -i narration.mp3 \
  -vf "drawtext=fontfile=font.ttf:textfile=cues.txt:fontcolor=white@0.85:\
  fontsize=48:x=(w-text_w)/2:y=(h-text_h)*0.6:\
  box=1:boxcolor=black@0.4:boxborderw=10" \
  -map 0:v -map 1:a -shortest -c:v libx264 -crf 22 -c:a aac output.mp4
```

### Breathing circle overlay (from pre-rendered GIF)
```bash
ffmpeg -stream_loop -1 -f lavfi -i "color=c=0x0a0a1a:s=1080x1920:r=30" \
  -stream_loop -1 -i breathing_circle.gif \
  -i narration.mp3 \
  -filter_complex "[1:v]scale=400:400[circle]; \
  [0:v][circle]overlay=x=(W-w)/2:y=(H-h)/2:shortest=1[out]" \
  -map "[out]" -map 2:a -shortest output.mp4
```

### ASS-based kinetic typography
```bash
ffmpeg -i audio.mp3 \
  -f lavfi -i "color=c=0x0d1117:s=1080x1920:d=60,format=yuv420p" \
  -vf "ass=kinetic_script.ass" \
  -map 1:v -map 0:a -shortest -c:v libx264 -crf 20 output.mp4
```
