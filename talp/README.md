# talp/ — Staging for Mining

**Purpose:** Staging area for source material before it is mined into atoms (HOOK, SCENE, STORY, REFLECTION, EXERCISE, INTEGRATION). Not ingested as-is; content here is mined per the Phoenix V4 pipeline.

**Contents:**

- **TE_\*** — Teacher series scripts (emotional labor, classroom management, workload, etc.). Same naming as `content/`; use for mining or as alternate/staged copy.
- **CP_\*** — Creative professional series (charging worth, creative block, rejection, etc.). Same naming as `content/`.
- **gen_alpha_*_complete.txt** — Gen Alpha scene library (anxiety, shame, grief, burnout, identity, loneliness, family conflict). Raw prose scenes for mining.
- **GEN_ALPHA_COMPLETE_SUMMARY.md** — Index/summary of the Gen Alpha scene set.
- **SYSTEMS_DOCUMENTATION.md** — Legacy systems doc (reference only). Canonical system spec: `specs/PHOENIX_V4_CANONICAL_SPEC.md`.

**Pipeline:** Material in this folder is mined → story/scene/exercise atoms → lint → candidate → approval → approved_atoms. See Canonical Spec Part 4 (Pipelines) and REPO_FILES.md.
