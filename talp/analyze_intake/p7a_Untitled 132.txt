do for shame: WRITER ONBOARDING — PHOENIX OMEGA V4
One Page. Read Before Writing.
WHAT YOU ARE WRITING
Story atoms. Not stories. Not lessons. Not therapy.
A story atom is:
* 60–150 words
* One person
* One moment
* One psychological pattern
* No teaching
If it explains, delete it. If it comforts, delete it. If the problem resolves, delete it.
THE FIVE QUESTIONS (ANSWER BEFORE DRAFTING)
1. Persona — Who is this for?
2. Engine — Which pattern is running?
3. Role — What job does this atom do?
4. Moment — What specific situation?
5. Turn — What shifts (if anything)?
If you can't answer all five, don't write.
ROLES (MEMORIZE THESE)
Role What It Does How It Ends Recognition "That's me" Inside the pattern, no shift Mechanism Proof Shows how pattern repeats Pattern visible, no escape Turning Point Crack in pattern Crack only, not cure Embodiment New behavior in motion Action taken, pattern quieter
Recognition and Mechanism Proof never improve. Turning Point and Embodiment never triumph.
ENGINES (CANONICAL SET — 7 ONLY)
Engine Core Question FALSE_ALARM Body reacting to danger that isn't there? SPIRAL One thought pulling the next? WATCHER Observing self from outside? OVERWHELM Too much input at once? SHAME Self collapsing after being seen? COMPARISON Value shifting by relative position? GRIEF Absence becoming present?
One engine per atom. No hybrids.
Deprecated (Do Not Use)
The following were early experiments, now superseded:
* BOUNDARIES → use SHAME or COMPARISON
* DEPRESSION → use WATCHER or GRIEF
* SELF_WORTH → use SHAME or COMPARISON
If you have existing batches with deprecated engines, they stand as-is. Do not write new batches with deprecated engines.
LANGUAGE BANS (AUTO-REJECT)
These words/phrases invalidate an atom:
* "anxiety" / "anxious" / "panic"
* "nervous system"
* "healing" / "journey"
* "finally realized"
* "the lesson was"
* "safe now"
* "what this shows"
No exceptions.
DISTRIBUTION (PER 20-ATOM BATCH)
Standard distribution:
Role Count Recognition 5 Mechanism Proof 5 Turning Point 5 Embodiment 5
Acceptable alternate (legacy batches): 6/6/4/4
Both distributions are valid. Pick one per batch. Do not mix within a batch.
SELF-CHECK (BEFORE SUBMITTING)
1. Is it 60–150 words?
2. One person, one moment?
3. Does the ending match the role?
4. No banned language?
5. Would a reader say "that's exactly what happens" (not "that helped me")?
If any answer is no, rewrite.
STRIP TEST (FOR PERSONA WORK)
Remove all persona-specific nouns. Replace with generic task nouns. If the engine weakens, the atom is wrong.
WHAT SUCCESS LOOKS LIKE
The reader recognizes the pattern. The reader does not feel taught. The reader does not feel better.
Recognition beats relief.
ONE RULE
Stories embody. Templates explain.
If your atom explains, delete it.
QUESTIONS?
* Engine confusion → See QA_7_ENGINE_CONFUSION_GUARD.md
* Role confusion → See WRITER_SPEC_ADDENDUM.md
* Calibration examples → See SPIRAL_CALIBRATION_ATOMS.md
Do not improvise. Follow the spec.
