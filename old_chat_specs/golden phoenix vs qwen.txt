Prose % Estimation (Teacher Share)
Simple word-based approach:

python
def _estimate_teacher_prose_share(self, article: ArticleContent) -> float:
    teacher_words = sum(
        s.word_count for s in article.sections.values()
        if s.has_teacher_content
    )
    total_words = article.total_word_count
    if total_words == 0:
        return 0.0
    return teacher_words / total_words
This uses SectionContent.has_teacher_content flag (set by assembler or test fixture). The validator doesn't parse prose — it trusts section metadata. This is the same approach as phoenix_v4 where validators check metadata, not raw text.

Exception: _check_teacher_first_person DOES scan text, using the same marker list from the existing article_assembler.py ("I teach", "I believe", "my practice", etc.).
