STEP 3 — LAYER STACK DEFINITION (DEEP DIVE)
What Step 3 really is
Step 3 is where you lock the creative degrees of freedom before any prose is generated.
It answers one question only:
What kinds of content are allowed to exist in this book — and where?
Not how well it’s written. Not what the words are. Only what layers are permitted to operate.
This is the step that prevents:
bloated chapters
mixed voices
accidental genre drift
“LLM did everything everywhere” output

Conceptually: what is a “layer”?
A layer is a type of content behavior that can appear in a chapter.
Examples:
narrative storytelling
conceptual explanation
somatic exercise instruction
meditation script
reflection prompts
editor-protected text
You do not want all layers active at once.

The Core Layer Stack (Canonical)
At minimum, every book has these possible layers:
Layer
Purpose
Narrative
Story, vignette, lived experience
Concept
Explanation, framing, insight
Exercise
Somatic / meditative instruction
Reflection
Prompts, journaling, integration
Editorial Freeze
Text that must not be regenerated
Step 3 decides which of these are ON, OFF, or LIMITED.

FILE 1: layer_stack.yaml (or .json)
This is the authoritative declaration of allowed layers.
Example:


layer_stack:
  narrative:
    enabled: false
    max_words_per_chapter: 0

  concept:
    enabled: true
    density: medium   # light | medium | heavy
    max_words_per_chapter: 1800

  exercise:
    enabled: true
    types_allowed:
      - somatic
      - meditative
    max_per_chapter: 1

  reflection:
    enabled: true
    max_prompts_per_chapter: 2

  editorial_freeze:
    enabled: true

What this does:
Prevents narrative creep if this is a practice-first book
Caps concept verbosity
Stops “3 exercises in one chapter”
Makes reflection intentional, not accidental
This file is loaded before outline generation.

FILE 2: template_profile.yaml
This ties template → layer stack defaults.
Example:


template_profile:
  name: "GenZ_30_Day_Reset"

  default_layers:
    narrative: off
    concept: light
    exercise: high
    reflection: light

  overrides_allowed:
    - exercise.max_per_chapter
    - reflection.max_prompts_per_chapter

Meaning:
This template cannot accidentally become narrative
Concept is intentionally light
Exercises are the core delivery mechanism
If a later step tries to inject narrative → ❌ blocked.

FILE 3: chapter_role_map.yaml
This prevents intra-chapter chaos.
Instead of every chapter doing everything, you define roles.
Example:


chapter_roles:
  - name: "Teach"
    layers_allowed: [concept]
  
  - name: "Practice"
    layers_allowed: [exercise]

  - name: "Integrate"
    layers_allowed: [reflection]

  - name: "Teach+Practice"
    layers_allowed: [concept, exercise]

Then chapters are assigned roles:


chapters:
  1: Teach
  2: Teach+Practice
  3: Practice
  4: Integrate

This alone eliminates:
overstuffed chapters
tone switching mid-chapter
accidental redundancy

FILE 4: exercise_policy.yaml
This is how your 71-exercise bank stays controlled.
Example:


exercise_policy:
  reuse_within_book: false
  max_exercises_per_chapter: 1

  allowed_by_layer_stack:
    somatic:
      allowed: true
    meditative:
      allowed: true
    thought_experiment:
      allowed: false

Now:
A Gen Z book won’t suddenly include a dense thought experiment
Meditations don’t stack unintentionally
Editors know violations are structural, not stylistic

FILE 5: editorial_freeze.yaml
This is critical and often missing in systems.
Example:


editorial_freeze:
  sections:
    - intro.promise
    - conclusion.integration
    - exercise.instructions

Meaning:
These sections must not be rewritten
Even if later passes run
Even if similarity checks fire
This protects:
carefully written intros
safety-reviewed exercises
platform-approved conclusions

# 4. IMPLEMENTATION

## 4.1 Core Validator Class (Updated)

python
# validator/governor_validator.py

import re
from difflib import SequenceMatcher
from typing import Dict, List, Tuple
from dataclasses import dataclass, field
from enum import Enum

class GovernorType(Enum):
    EXPLAIN_CAP = "explain_cap"
    DEMO_SATURATION = "demo_saturation"
    DUPLICATION_GUARD = "duplication_guard"
    TRUNCATION_GUARD = "truncation_guard"

@dataclass
class Violation:
    governor: GovernorType
    beat_id: str
    template_path: str
    line_number: int
    severity: str  # "FAIL" or "WARN"
    message: str
    insight_id: str = ""
    current_count: int = 0
    max_allowed: int = 0
    duplicate_of: str = ""
    similarity: float = 0.0
    pattern_matched: str = ""
    text_fragment: str = ""

@dataclass
class GovernorConfig:
    explain_cap_max: int = 1
    demo_saturation_max: int = 3
    demo_saturation_strict: bool = False
    duplication_similarity_threshold: float = 0.80
    duplication_min_lines: int = 3
    truncation_patterns: List[str] = field(default_factory=lambda: [
        r'\b\w{1,2}\.$',
        r'\s+\.$',
        r'[a-z]{1,3}\s*$',
    ])

class GovernorValidator:
    def __init__(self, config: GovernorConfig, template_path: str):
        self.config = config
        self.template_path = template_path
        self.violations: List[Violation] = []
        
        # Tracking counters
        self.explain_counts: Dict[str, Dict[int, int]] = {}  # insight_id -> {chapter: count}
        self.demo_counts: Dict[str, Dict[int, int]] = {}
        self.rendered_beats: Dict[int, List[Tuple[str, str]]] = {}  # chapter -> [(beat_id, text)]
    
    def validate_explain_cap(self, beat_id: str, chapter: int, 
                             insight_id: str, line_number: int) -> None:
        """Governor A: Explain Cap"""
        chapter_counts = self.explain_counts.setdefault(insight_id, {})
        current = chapter_counts.get(chapter, 0)
        
        if current >= self.config.explain_cap_max:
            self.violations.append(Violation(
                governor=GovernorType.EXPLAIN_CAP,
                beat_id=beat_id,
                template_path=self.template_path,
                line_number=line_number,
                insight_id=insight_id,
                current_count=current + 1,
                max_allowed=self.config.explain_cap_max,
                severity="FAIL",
                message=f"EXPLAIN cap exceeded: '{insight_id}' already explained "
                        f"{current}x in Ch{chapter} (max: {self.config.explain_cap_max})"
            ))
        else:
            chapter_counts[chapter] = current + 1
    
    def validate_demo_saturation(self, beat_id: str, chapter: int,
                                  insight_id: str, line_number: int) -> None:
        """Governor B: Demo Saturation"""
        chapter_counts = self.demo_counts.setdefault(insight_id, {})
        current = chapter_counts.get(chapter, 0)
        
        if current >= self.config.demo_saturation_max:
            severity = "FAIL" if self.config.demo_saturation_strict else "WARN"
            self.violations.append(Violation(
                governor=GovernorType.DEMO_SATURATION,
                beat_id=beat_id,
                template_path=self.template_path,
                line_number=line_number,
                insight_id=insight_id,
                current_count=current + 1,
                max_allowed=self.config.demo_saturation_max,
                severity=severity,
                message=f"DEMONSTRATE saturation: '{insight_id}' has "
                        f"{current + 1} demos in Ch{chapter} (max: {self.config.demo_saturation_max})"
            ))
        
        chapter_counts[chapter] = current + 1
    
    def validate_duplication(self, beat_id: str, chapter: int,
                             rendered_text: str, line_number: int) -> None:
        """Governor C: Duplication Guard"""
        chapter_beats = self.rendered_beats.setdefault(chapter, [])
        
        # Check against all previous beats in this chapter
        for prev_beat_id, prev_text in chapter_beats:
            if len(prev_text.split('\n')) < self.config.duplication_min_lines:
                continue
            if len(rendered_text.split('\n')) < self.config.duplication_min_lines:
                continue
                
            similarity = SequenceMatcher(None, prev_text, rendered_text).ratio()
            
            if similarity >= self.config.duplication_similarity_threshold:
                self.violations.append(Violation(
                    governor=GovernorType.DUPLICATION_GUARD,
                    beat_id=beat_id,
                    template_path=self.template_path,
                    line_number=line_number,
                    duplicate_of=prev_beat_id,
                    similarity=round(similarity, 2),
                    severity="FAIL",
                    message=f"Duplicate block detected: {int(similarity*100)}% similar to {prev_beat_id}"
                ))
                return  # One duplicate finding is enough
        
        # Add this beat to the chapter's rendered beats
        chapter_beats.append((beat_id, rendered_text))
    
    def validate_truncation(self, beat_id: str, rendered_text: str, 
                            line_number: int) -> None:
        """Governor D: Truncation Guard"""
        for pattern in self.config.truncation_patterns:
            match = re.search(pattern, rendered_text)
            if match:
                # Extract context around the match
                start = max(0, match.start() - 20)
                end = min(len(rendered_text), match.end() + 10)
                fragment = rendered_text[start:end]
                
                self.violations.append(Violation(
                    governor=GovernorType.TRUNCATION_GUARD,
                    beat_id=beat_id,
                    template_path=self.template_path,
                    line_number=line_number,
                    pattern_matched=pattern,
                    text_fragment=fragment,
                    severity="FAIL",
                    message=f"Truncation detected: '{fragment}'"
                ))
                return  # One truncation finding is enough
    
    def get_failures(self) -> List[Violation]:
        return [v for v in self.violations if v.severity == "FAIL"]
    
    def get_warnings(self) -> List[Violation]:
        return [v for v in self.violations if v.severity == "WARN"]
    
    def has_failures(self) -> bool:
        return len(self.get_failures()) > 0
    
    def generate_report(self) -> dict:
        """Generate structured report for CI/CD integration"""
        failures = self.get_failures()
        warnings = self.get_warnings()
        
        return {
            "validation_result": "FAIL" if failures else "PASS",
            "total_failures": len(failures),
            "total_warnings": len(warnings),
            "governors": {
                "explain_cap": {
                    "failures": [self._violation_to_dict(v) for v in failures 
                                 if v.governor == GovernorType.EXPLAIN_CAP]
                },
                "demo_saturation": {
                    "warnings": [self._violation_to_dict(v) for v in warnings
                                 if v.governor == GovernorType.DEMO_SATURATION],
                    "failures": [self._violation_to_dict(v) for v in failures
                                 if v.governor == GovernorType.DEMO_SATURATION]
                },
                "duplication_guard": {
                    "failures": [self._violation_to_dict(v) for v in failures
                                 if v.governor == GovernorType.DUPLICATION_GUARD]
                },
                "truncation_guard": {
                    "failures": [self._violation_to_dict(v) for v in failures
                                 if v.governor == GovernorType.TRUNCATION_GUARD]
                }
            }
        }
    
    def _violation_to_dict(self, v: Violation) -> dict:
        d = {
            "beat_id": v.beat_id,
            "template_path": v.template_path,
            "line_number": v.line_number,
            "severity": v.severity,
            "message": v.message
        }
        if v.insight_id:
            d["insight_id"] = v.insight_id
            d["current_count"] = v.current_count
            d["max_allowed"] = v.max_allowed
        if v.duplicate_of:
            d["duplicate_of"] = v.duplicate_of
            d["similarity"] = v.similarity
        if v.pattern_matched:
            d["pattern_matched"] = v.pattern_matched
            d["text_fragment"] = v.text_fragment
        return d
