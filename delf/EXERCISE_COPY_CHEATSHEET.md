# Landing Page Copy — All Exercises
## One page per exercise. Change 4 fields. Done.

---

### HOW TO USE
Copy TEMPLATE_landing.html. Change:
1. `<title>` — Exercise Name + Free Tool
2. `h1` — Exercise Name (split over 2 lines for impact)
3. `.benefit` — One italic line. What they'll *feel* after.
4. `.duration` — From the table below
5. `TOOL_URL` — Link to the actual interactive tool
6. `ML_GROUP_ID` — Unique MailerLite group per exercise
7. `EXERCISE_TAG` — Snake_case tag for segmentation
