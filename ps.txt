═══════════════════════════════════════════
STEP 2: PREFLIGHT
═══════════════════════════════════════════

Before any branch, commit, push, PR, merge, or workflow action:

1. `git branch --show-current`
2. `git status --short`
3. `git fetch origin`
4. `git rev-list --left-right --count origin/main...HEAD`
5. `PYTHONPATH=. python scripts/git/push_guard.py`
6. `scripts/ci/preflight_push.sh`
7. `bash scripts/git/health_check.sh`

Never skip preflight.

═══════════════════════════════════════════
STEP 3: BRANCH RULES
═══════════════════════════════════════════

Always branch from `origin/main`:

```bash
git fetch origin
git checkout -b agent/<task-summary> origin/main
```

Never branch from `codex/*` or another local branch.
If push-guard blocks because the branch base is wrong, recover with:

```bash
git fetch origin
git checkout -b agent/<task-summary>-clean origin/main
git cherry-pick <commit>
git push -u origin agent/<task-summary>-clean
```

═══════════════════════════════════════════
STEP 4: HOURLY CHECKLIST
═══════════════════════════════════════════

Run this every hour while doing repo work:

1. Confirm branch and working tree are sane
2. Confirm no `.git/index.lock`
3. Confirm you are still near `origin/main`
4. Confirm push-guard still passes
5. Confirm no oversized files slipped in
6. Confirm no stale cherry-pick/rebase is in progress
7. Confirm planned push still fits PR scope
8. Confirm next action: branch, commit, push, PR, or stop

Command:

```bash
bash scripts/git/health_check.sh
```
