---
family: F1
voice_mode: teacher
mechanism_emphasis: automatic
---
The mechanism is metacognitive monitoring that's recursive and never terminates. The base case never arrives. The call stack gets infinitely deep. I make a comment in a meeting. Did I say something stupid? That calls what_might_they_think(). Which calls what_does_this_mean_for_my_job(). Which calls what_will_I_do_if_I_lose_my_job(). And somewhere around recursive depth 20, the system crashes into anxiety.
The mechanism is automatic. I'm not choosing to overthink. The system is configured to believe that if I think through every branch, every scenario, every possible outcome, I can achieve control over something fundamentally uncontrollable. So the thinking continues. The recursive depth keeps increasing. I can see that I'm not generating new insight, just cycling through the same variables in different combinations. And I can't stop it. The mechanism keeps running because stopping feels like losing control.
