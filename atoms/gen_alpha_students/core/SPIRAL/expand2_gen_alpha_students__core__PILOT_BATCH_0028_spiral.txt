## SPIRAL / MECHANISM_PROOF v03

Jaden checks his grades. One B. That becomes: I'm slipping. Which becomes: I've probably been slipping for a while. Which becomes: What if I can't keep up? Which becomes: What if I was never actually smart? He refreshes the page. The B is still there. Nothing has changed except the story he's building. Each layer makes the B heavier. By 10pm, the B feels like proof of something permanent.
