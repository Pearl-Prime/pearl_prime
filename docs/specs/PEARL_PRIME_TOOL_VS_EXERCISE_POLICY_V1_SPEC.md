# Pearl Prime Tool vs Exercise Policy V1

**Status:** Canonical doctrine amendment.

## Distinction

| Tool | Exercise |
|---|---|
| Diagnostic, decision rule, stop rule, checklist, or short application | Five-layer guided practice |
| Brief; usable immediately after an idea becomes clear | Deliberate embodied sequence |
| `AUTHOR_TOOL_INTRO` → `PRACTICE_APPLICATION_TOOL` / `TOOL` | EXERCISE slot with a canonical practice ID |

Tools never expand into pseudo-exercises. Exercises never collapse into tips or worksheets without the required five layers. A chapter may contain both when their jobs are distinct.

## Placement and scale

- Place the tool after the governing idea/mechanism is understandable.
- Keep exercises on EXERCISE slots, including §§4/8 where the chapter architecture calls for them.
- Target 150–250 reusable, persona-adaptable tools for a 1,000-book catalog; do not force tool inventory to match the 311 practice count.
- Validate job, placement, brevity, and non-duplication independently from exercise integrity.
