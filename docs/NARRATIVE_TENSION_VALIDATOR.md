# Narrative Tension Validator

**Prevents flat arcs.** Band diversity ≠ narrative momentum. This validator ensures tension progression. Lightweight, human-usable.

---

## Check the Emotional Slope

Across chapters, does tension:

- **A)** Rise  
- **B)** Break  
- **C)** Resolve  

Or does it:

- Oscillate  
- Plateau  
- Stay neutral  

If the emotional intensity graph looks like:

`3 → 3 → 3 → 3 → 3`

even if compliant — it's boring.

---

## Chapter-Level Tension Questions

Each chapter should answer **at least one**:

- What is at risk?  
- What is misunderstood?  
- What belief is being challenged?  
- What discomfort is increasing?  

If a chapter feels like *calm explanation + calm exercise*, it needs friction.

---

## Practical Use

1. Print the band sequence (e.g. from compiled plan: `dominant_band_sequence` or `emotional_temperature_sequence`).  
2. Ask: **Where does rupture happen?**  
3. If nowhere → arc is weak.
