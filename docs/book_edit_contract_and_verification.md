## Generator / gating (secondary)

- One canonical **uncertainty-as-alarm** block per book; later chapters: ≤2 sentences or cross-reference.
- Ban template phrases addressing **the chapter** to the reader.
- Cap **scene families** per manuscript; require **opener-to-scene** pairing in outline before generation.
- **Bright** (or equivalent): one plain-English introduction; thereafter sparse, prefer somatic vocabulary.
