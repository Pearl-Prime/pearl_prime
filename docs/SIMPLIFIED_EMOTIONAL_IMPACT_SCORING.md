# Simplified Emotional Impact Scoring Model

**Non-AI, manual first pass.**  
You are currently optimizing structure. This checks impact.

---

## Rule

Each chapter must contain **at least 2** of the following:

| Moment            | Feels like                          |
|-------------------|-------------------------------------|
| Recognition       | "That's me."                        |
| Reframe           | "I've never thought of it that way."|
| Relief            | "Oh… that makes sense."             |
| Challenge         | "I need to change that."            |
| Identity          | "Maybe I'm not broken."             |

If a chapter contains **zero** of these, it is structurally valid but **emotionally inert**.

Band diversity does **not** guarantee these moments.
