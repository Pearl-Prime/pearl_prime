# Teachers — source and reference materials

Per-teacher **source** materials (intake, reference docs) live here. Pipeline **runtime** content lives in `SOURCE_OF_TRUTH/teacher_banks/<teacher_id>/`. **Config** (registry, per-teacher YAML) lives in `config/teachers/`.

| Teacher   | Folder           | Runtime bank                    | Config              |
|----------|------------------|----------------------------------|---------------------|
| Ahjan    | **ahjan/**       | SOURCE_OF_TRUTH/teacher_banks/ahjan | config/teachers/ahjan.yaml |
| Master Wu, Sai Maa, etc. | master_wu/, sai_ma/, … | (when added) | config/teachers/teacher_registry.yaml |

See **teachers/ahjan/README.md** for Ahjan layout and intake script.
